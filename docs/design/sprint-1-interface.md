# Interface Sprint 1

Écrans requis : sélection des héros, salle début de tour, héros sélectionné avec cases atteignables, cible attaquable, tour ennemi, victoire, défaite et reprise.

Direction : fantasy de taverne sombre mais lisible, bois, cuir, cuivre et parchemin. Les jetons restent temporaires. La priorité est tactile : zones de 44 × 44 px, un doigt, libellés français et contraste lisible.

Figma : connecteur indisponible dans cet environnement ; cette spécification sert de contenu à reporter dans Figma.
