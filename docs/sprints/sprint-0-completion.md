# Sprint 0 — Rapport de clôture

## Livré

- PWA Vite + TypeScript installable ;
- rendu PixiJS adaptatif ;
- moteur d’événements déterministe ;
- sauvegarde IndexedDB versionnée ;
- schémas Zod et validation du pack Bastognac ;
- tests unitaires et smoke tests Playwright exécutés sur le build de production servi par Vite Preview ;
- CI GitHub et déploiement Pages ;
- Dependabot ;
- documentation, ADR, modèles GitHub et diagramme FigJam ;
- design system initial dans Figma.

## Définition de terminé

Le Sprint 0 est terminé lorsque la Pull Request est verte, fusionnée dans `main`, et que le workflow Pages publie l’écran de fondation. La PR #3, empilée sur `sprint-0/foundations`, doit être verte et fusionnée avant la PR #2 afin que la CI complète soit validée sur la pile.

## Dette acceptée

- icône PWA provisoire en SVG ;
- contenu Bastognac limité à un manifeste et une définition minimale ;
- absence d’audio réel ;
- aucun gameplay de combat avant le Sprint 1 ;
- aucun appel OpenAI depuis le client, ni aucune clé OpenAI dans les sources, le lockfile, les workflows, le build, les sourcemaps ou les manifestes PWA.
